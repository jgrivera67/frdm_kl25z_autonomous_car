# Autonomous car with the FRDM-KL25Z board

See [slides](doc/project_overview.pdf).

To be able to flash firmware on an NXP KL25Z microcontroller install
a recent version of the J-Link OpenSDA boot loader firmware on the
device following the steps described at
https://www.segger.com/products/debug-probes/j-link/models/other-j-links/opensda-sda-v2/.

J-Link OpeSDA bootloader firmware for the KL25Z can be found at:
https://www.segger.com/downloads/jlink/#JLinkOpenSDABoardSpecificFirmwares
https://www.segger.com/downloads/jlink/OpenSDA_FRDM-KL25Z
